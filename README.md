# React Recipe Book

```
npm i -D babel-core babel-loader babel-preset-es2015 babel-preset-react css-loader sass-loader node-sass style-loader webpack
npm i -S react react-dom
```
Then run
```
webpack
```
Then open  `/public/index.html` in browser.
